# 水利可视化设计原则

> 图表选型决策 + 设计原则 + 可访问性。**方法论层**——决定"画什么图、怎么画得准"；
> 具体 9 种水利图表的代码模板见 `water-visualization/references/chart_templates.md`。

## 一、按数据关系选图

| 要表达的关系 | 推荐图 | 水利示例 |
|-------------|--------|---------|
| **随时间变化（趋势）** | 折线图 | 站水位过程线、月降雨趋势 |
| **跨类别比较** | 竖向柱状图 | 各站累计降雨对比 |
| **排名** | 横向柱状图 | 雨量站累计降雨排名 |
| **整体构成** | 堆叠柱状图 | 各水质等级断面占比 |
| **构成随时间变化** | 堆叠面积图 | 汛期各等级水质断面比例演变 |
| **分布** | 直方图 / 箱线图 | 全市水位值分布、各河段 DO 分布对比 |
| **两变量相关** | 散点图 | 降雨量 vs 水位抬升 |
| **多变量相关** | 相关矩阵热力图 | 水质 4 指标相关性 |
| **地理分布** | 分级着色地图 | 各行政区降雨/水位空间分布 |
| **实测 vs 预测** | 双线对比图 | 预测水位与实测水位叠加 + 偏差带 |
| **多 KPI 同屏** | 小多图（small multiples） | 各重点站水位微缩趋势阵列 |
| **状态汇总** | 色块/状态矩阵 | 各站预警状态（绿/黄/红）面板 |

## 二、水利专用图表要点

| 图表 | 关键设计 |
|------|---------|
| **水位过程线** | 折线 + 叠加警戒水位(WRZ)/保证水位(GRZ) 水平虚线；Y 轴单位 m，保留 2 位小数 |
| **降雨柱状图** | 按雨强分级着色（小雨蓝/中雨橙/大雨红）；日柱状，X 轴日期 |
| **水质多指标** | DO/CODMn/NH3N/TP 四子图（small multiples），共享 X 轴时间，各自 Y 轴量纲 |
| **预警状态面板** | 绿=正常 / 黄=超警戒 / 红=超保证；色块矩阵，行=站，列=时间 |
| **预测 vs 实测** | 双折线 + 偏差阴影带；标注最大偏差点 |
| **水库调度图** | 水位过程线 + 入库/出库流量双轴（慎用双轴，见下）+ 汛限/正常蓄水位参考线 |

## 三、何时不该用某些图

- **饼图**：除非 <6 类且只需粗略比例，否则别用——人眼不擅比较角度。**改用柱状图**。
- **3D 图**：永远不要。扭曲感知、不增信息。
- **双轴图**：慎用，易暗示虚假相关。若必须用，两轴都清晰标注、用颜色区分左右轴。
- **堆叠柱（多类别）**：中间段难比较。改用小多图或分组柱状。
- **仅用红/绿区分**：8% 男性红绿色盲。**改用蓝/橙为主对** + 线型/图案辅助。

## 四、设计原则

### 颜色
- **颜色承载信息**，不是装饰：用一个亮色突出关键洞察，其余灰化。
- **顺序数据**（水位高低、雨强）：单色渐变（浅→深）。
- **发散数据**（距平、偏差）：双色渐变 + 中性中点。
- **分类数据**：异色，最多 6-8 色否则混乱。
- **色盲友好调色板**：`['#4C72B0','#DD8452','#55A868','#C44E52','#8172B3']`（蓝/橙为主对）。

### 字体/标注
- **标题讲洞察**："7月降雨较常年偏多 35%" 胜过 "月降雨量"。
- **副标题加上下文**：时间范围、过滤条件、数据来源。
- **轴标签可读**：避免 90° 旋转；缩短或换行。
- **关键点加数值标注**，不必每个柱都标。

### 布局
- **去 chartjunk**：去掉不承载信息的网格线/边框/背景。
- **按值排序**：类别按值排（非字母序），除非有自然序（月份、流程）。
- **合适长宽比**：时间序列宽于高（3:1~2:1）；比较图可方一些。
- **留白**：别挤，给每个图呼吸空间。

### 准确性
- **柱状图 Y 轴从 0 开始**（必须）。从 95 画到 100 会夸大 5% 差异。
- **折线图可非零起点**：当变化范围本身有意义时（如水位在 3-5m 波动）。
- **多图比较时统一坐标范围**。
- **标注不确定性**：误差棒/置信区间/范围带（如预测区间）。
- **标注单位**：水位 m、降雨 mm、流量 m³/s、水质 mg/L——别让读者猜。

## 五、可访问性

### 色盲
- 永不只靠颜色区分系列——加图案填充、不同线型（实/虚/点）、直接标注。
- 用色盲模拟器检查（Coblis / Sim Daltonism）。
- 用 `sns.color_palette("colorblind")`。

### 屏幕阅读器/通用
- 图加 **alt 文本**描述关键发现。
- 关键图旁附**数据表**替代。
- 文字最小 10pt（标注）/ 12pt（标题）。
- 考虑黑白打印是否仍可辨。

### 交付前检查清单
- [ ] 去掉颜色后仍可区分系列（图案/线型/标注）
- [ ] 标准缩放下文字可读
- [ ] 标题讲洞察，不只是指标名
- [ ] 坐标轴标注了单位（m / mm / m³/s / mg/L）
- [ ] 图例清晰、不遮挡数据
- [ ] 标注数据来源与时间范围
- [ ] 中文正常显示（非方块 □□□——需 CJK 字体：`apt install fonts-noto-cjk`）
